# Privacy Policy — UFC Spoiler Blocker for Paramount+

**Last updated:** 2026-03-31

## Data Collection

This extension does **not** collect, transmit, or share any user data. No personal information, browsing history, or usage analytics are gathered at any point.

## Local Storage

The only data stored by this extension is two boolean preference flags (`hideDuration` and `replaceImages`) saved locally on the user's device via `chrome.storage.local`. This data:

- Never leaves the user's device
- Is never sent to any server or third party
- Can be cleared at any time by removing the extension

## Permissions Used

| Permission | Reason |
|---|---|
| `storage` | Save the user's on/off preferences between sessions |
| `*://*.paramountplus.com/*` | Modify the DOM of Paramount+ pages to hide duration labels and replace thumbnails |

No other permissions are requested or used.

## Third-Party Services

This extension does not integrate with, communicate with, or send data to any third-party service.

## Contact

If you have any questions about this privacy policy, please open an issue in the extension's repository.
